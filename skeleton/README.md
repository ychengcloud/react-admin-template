# 全新技术栈管理系统

全新技术栈的后台管理系统

- React
- Recoil
- React Query
- React Hooks
- Vite
- Typescript

## 安装使用

- 获取项目代码

> git clone https://github.com/ychengcloud/react-vite-admin.git

- 安装依赖

> cd react-vite-admin

> yarn install

- 运行

> yarn dev

- 打包

> yarn build
